# Trackit (Work in progress)

This repository contains the Trackit web application. The project is currently under active development and not yet production-ready.

Quick reference

- Status: In active development
- Tech: Next.js (App Router), TypeScript, Tailwind CSS, Prisma, tRPC

Run locally

1. Install dependencies:

   pnpm install

2. Start the dev server:

   pnpm run dev

That's it — this is a small reference README. For more details, check the `src/` folder and the project documentation.
